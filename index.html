Cómo escribir las funciones

  • Potencias: x^2, x^n
  • Raíz cuadrada: sqrt(x)
  • Fracciones: (x^2-1)/(x-1)
  • Trigonometría: sin(x), cos(x), tan(x)
  • Logaritmos: log(x) → natural, log10(x) → base 10, log(x, n) → base n
  • Exponenciales: exp(x) → e^x
