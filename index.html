Derivadas Generales

Introduce una función y calcula su derivada respecto a x.

Cómo escribir tus funciones

  • Para potencias usa ^, ejemplo: x^2
  • Para raíces usa sqrt(x)
  • Logaritmos: log(x) para base e, log10(x) para base 10, log2(x) para base 2
  • Funciones trigonométricas: sin(x), cos(x), tan(x)
  • Exponencial: exp(x)

Usa el autocompletado mientras escribes para ver sugerencias de funciones.
